# Provenance Guard

A backend system that classifies whether submitted text-based creative work was
written by a human or generated by AI, scores confidence in that classification,
surfaces a plain-language transparency label, and lets creators appeal — with
rate limiting and a structured audit log.

## Setup

```bash
python -m venv .venv
source .venv/bin/activate            # Mac/Linux
pip install -r requirements.txt
cp .env.example .env                 # then add your GROQ_API_KEY
python app.py                        # serves on http://localhost:5000
```

## API

| Method | Route | Body | Returns |
| --- | --- | --- | --- |
| POST | `/submit` | `{"text": ..., "creator_id": ...}` | `{content_id, attribution, confidence, label}` |
| POST | `/appeal` | `{"content_id": ..., "creator_reasoning": ...}` | `{content_id, status, message}` |
| GET | `/log` | — | `{"entries": [...]}` |

```bash
curl -s -X POST http://localhost:5000/submit \
  -H "Content-Type: application/json" \
  -d '{"text": "The sun dipped below the horizon.", "creator_id": "test-user-1"}' \
  | python -m json.tool
```
